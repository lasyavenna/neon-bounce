# Neon Bounce 🎮

A 2D platformer demo built with **Godot 4** featuring neon-styled visuals, player movement, physics, collectibles, and enemies.

## Features
- Smooth player movement with gravity and jumping
- Neon cyan visual style
- Score system with UI
- Enemy AI that patrols back and forth
- Coin collectibles
- Built with GDScript

## How to Play
- **Arrow Keys** — Move left/right
- **Space / Up Arrow** — Jump

## Requirements
- Godot Engine 4.x

## Setup
1. Clone the repo
2. Open Godot 4
3. Click "Import" and select the `project.godot` file
4. Press Play ▶
## Built With
- [Godot Engine 4](https://godotengine.org/)
